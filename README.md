# 🔮 Nebula 天機占卜 API (Divination FastAPI)

這是一個基於 **FastAPI** 開發的個人化運勢占卜系統。使用者輸入姓名與生日後，系統會結合西洋星座、生肖與隨機運算，產生每日運勢分析，包含幸運色、幸運數字以及針對「感情、事業、學業、財運」的詳細建議。

## ✨ 功能特色

* **自動計算星座與生肖**：根據輸入的生日自動判斷。
* **每日運勢生成**：提供 1-5 星的運勢評分與幸運小語。
* **客製化選項**：使用者可勾選想要詳細了解的運勢面向（如：感情、事業）。
* **響應式前端介面**：內建美觀的 HTML/CSS 前端網頁，支援手機與電腦瀏覽。
* **RESTful API**：提供標準 JSON 格式的 API 接口，方便前後端分離開發。

## 🛠️ 技術堆疊 (Tech Stack)

* **Backend**: Python 3.10+, FastAPI
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
* **Deployment**: 支援 Render / Heroku 等雲端平台部署
* **Libraries**: `uvicorn`, `pydantic`
